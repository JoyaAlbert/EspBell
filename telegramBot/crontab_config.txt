# Configuración de crontab para el bot de Telegram
# 
# Para aplicar esta configuración:
# 1. Ejecuta: crontab -e
# 2. Copia las siguientes líneas al final del archivo
# 3. Guarda y sal del editor
#
# IMPORTANTE: Reemplaza /home/joya con tu ruta real si es diferente

# Encender el bot a las 8:00 AM todos los días
0 8 * * * /home/joya/Documentos/EspBell/telegramBot/start_bot.sh

# Apagar el bot a las 11:00 PM (23:00) todos los días  
0 23 * * * /home/joya/Documentos/EspBell/telegramBot/stop_bot.sh

# Opcional: Verificar estado cada hora entre 8 AM y 11 PM
# 0 8-22 * * * /home/joya/Documentos/EspBell/telegramBot/status_bot.sh >> /home/joya/Documentos/EspBell/telegramBot/status.log

# Formato del crontab:
# minuto(0-59) hora(0-23) día_mes(1-31) mes(1-12) día_semana(0-7, 0=domingo)
#
# Ejemplos adicionales:
# - Solo días laborables: 0 8 * * 1-5 (lunes a viernes)
# - Solo fines de semana: 0 8 * * 0,6 (domingo y sábado)
# - Cada 30 minutos: */30 * * * *
